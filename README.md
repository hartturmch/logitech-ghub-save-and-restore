# Logitech G HUB Backup/Restore

A simple Windows tool to back up and restore Logitech G HUB settings, including profiles, macros, DPI, lighting, and per-application configuration.

## How to use

1. Double-click `GHubBackupRestore.bat`.
2. Choose your language.
3. Use the menu to create a backup, restore a backup, or delete old backups.

Backups are created in the same folder as the script using this format:

```text
GHubConfig-Backup-YYYY-MM-DD_HH-mm-ss
```

To restore, choose a restore option in the menu and confirm when prompted.
